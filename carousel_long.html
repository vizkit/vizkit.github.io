Phase 1 : Presenting
Phase 1.1

Expliquer l'atelier

Phase 1.2

Il faut donner le contexte des datasets ainsi que les datasets. Par exemple si le data présente des données d'un compte bancaire on peut expliquer au groupe qu'ils doivent aider un ami virtuel qui a besoin d'aide par rapport à ses dépenses

Phase 2 : Authoring
Phase 2.1

Les groupes sont maintenant inviter à créer une première visualisation avec le dataset qu'ils ont reçu.

Phase 2.2

Les groupes vont alors se demander au sein du groupe comment faire pour représenter au mieux les datas.

Phase 3 : Reading

Donner à présent les feuilles questionnaires et personae à chaque groupe. Cela devrait leur permettre de comprendre les traits importants des données qu'ils ont réussi à rendre compte dans leur visualisation et ceux qu'ils n'ont pas réussi. Les personae doivent leur donner une vision plus pratique des données qu'ils ont manipulées.

Phase 4 : Editing
Phase 4.1

En reprenant les remarques qu'ils se sont faites à l'étape précédente, les groupes peuvent à présent reprendre leur visualisation et l'améliorer

Phase 4.2

Vous pouvez à présent leur donner des stylos, des post-its ou tout matériel qui peut leur permettre d'améliorer leur visualisation.

Phase 5 : Critiquing
Phase 5.1

Donner les feuilles d'évaluation des autres groupes.

Phase 5.2

Chaque groupe choisit un évaluateur et un visiteur parmi eux. Les visiteurs doivent alors se déplacer sur une autre table. quelles sont les informations qu'il arrive à retirer de la visualisation

Phase 5.3

Chaque visiteur explique à l'évaluateur de la table sur laquelle il se trouve qu'il voit.

Phase 5.4

Les évaluateurs font des croix dans le tableau qu'on leur a donné pour savoir quels sont les éléments qui ont été trouvés et ceux qui ne l'ont pas été.

Phase 6 : Conclusion
Phase 6.1

Laisser les groupes discuter de leurs différentes visualisations. Dire alors que certains groupes ont les mêmes données et leur faire remarquer les différentes visualisations possibles à partir d'un même set de données.

Phase 6.2

Présenter alors différentes visualisations possibles pour le type de set de données sur lesquelles ils ont travaillés mais aussi de manière plus général.

Phase PI

Bravo! Vous avez réussi à sensibiliser vos participants à l'importance de la visualisation de données. Vous pouvez maintenant retrouver ce guide d'utilisation sous forme de pdf ici
